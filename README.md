# Agentic Systems / Decode Travel 2025 Hackathon

Repository providing travel-related data services (via MCP servers) for participants of the Decode Travel 2025 hackathon in Barcelona.

Official Hackathon Website: [Decode Travel Barcelona 2025](https://decode.travel/barcelona2025/)


## Purpose
Offer:
- Access to curated MCP servers exposing travel-related content (details TBD).
- Lightweight examples to integrate these servers into hackathon projects.
- On-site and Discord support.

Use is optional; teams may combine with their own data sources.

## Access
1. Request access from a mentor during the event.
2. Install an MCP-compatible client or SDK (Node, Python).
3. Point client to server base URL (to be announced).
4. Start querying endpoints or invoking MCP tools.

## Guidelines
- Cache responsibly.
- Do not attempt to re-identify individuals (data is aggregated).
- Respect rate limits announced on site.
